# Personal Diary – Java

This is a simple Personal Diary application developed using Java.
It allows the user to write daily diary entries and read them later.
The diary is private and protected with a password.

## Features
- Write diary entries like daily pages
- Add new diary content without deleting old entries
- Read previously written diary entries
- Password-protected (only you can access)
- Simple console-based program

## How to Use
1. Run the program
2. Enter the diary password
3. Choose:
   - Write diary to add new entries
   - Read diary to view all entries
4. Type END to save diary content

## Files
- Diary.java – main Java program
- myDiary.txt – diary content file (auto-created)
- README.md – project information

## Technologies Used
- Java
- File Handling
- Console Application
